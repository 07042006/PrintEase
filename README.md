# PrintEase
PrintEase: Extensão Automatizada de Impressão no Chrome

## Configurações para impressões
### Tamanho do papel
```javascript
paperSizeOptions = {
    "options": [
        "Carta", "Tablóide", "Extrato", "Executivo", "A3", "A4", "A5", "B4 (JIS)", "B5 (JIS)"
    ]
}
```

### Páginas por folha
```javascript
pagePerSheet = {
    "options": [
        1, 2, 4, 6, 9, 16
    ]
}
```

### Margens
```javascript
marginsOptions = {
    "options": [
        "Padrão", "Nenhuma", "Personalizada"
    ]
}
```
